# 常见症状诊疗指南

## 一、发热

### 定义
体温超过37.3℃（腋温）为发热。
- 低热：37.3～38.0℃
- 中等热：38.1～39.0℃
- 高热：39.1～41.0℃
- 超高热：>41.0℃

### 常见病因
**感染性发热（最常见）：**
- 细菌感染：肺炎、尿路感染、扁桃体炎、败血症
- 病毒感染：流感、新冠、EB病毒感染（传染性单核细胞增多症）
- 真菌感染：免疫功能低下患者常见

**非感染性发热：**
- 自身免疫病：系统性红斑狼疮、类风湿关节炎
- 恶性肿瘤：淋巴瘤（典型表现为长期低热）
- 药物热：抗生素、抗癫痫药等
- 中枢性发热：脑外伤、脑出血后

### 处理原则
- 体温<38.5℃：多饮水、物理降温（温水擦浴）
- 体温≥38.5℃：对乙酰氨基酚 500mg 或布洛芬 400mg，4～6小时后可重复
- **以下情况需立即就医：**
  - 超高热（>41℃）
  - 发热伴皮疹（考虑药疹、麻疹、猩红热）
  - 发热伴颈项强直（警惕脑膜炎）
  - 发热伴意识障碍
  - 持续发热>3天，原因未明

## 二、咳嗽

### 按病程分类
- **急性咳嗽**（<3周）：最常见为感染后咳嗽、普通感冒、急性气管支气管炎
- **亚急性咳嗽**（3～8周）：感染后咳嗽（最常见）、百日咳
- **慢性咳嗽**（>8周）：鼻后滴漏综合征、咳嗽变异型哮喘、胃食管反流病（GERD）

### 各类型要点
**干咳（无痰）：**
- 感染初期
- 咳嗽变异型哮喘（常见慢性干咳原因，夜间加重）
- ACEI类降压药副作用（服药后数天至数月出现）
- 胃食管反流（夜间、卧位加重）

**湿咳（有痰）：**
- 白色黏痰：慢性支气管炎、哮喘急性发作
- 黄绿色脓痰：细菌感染（支气管炎、肺炎）
- 铁锈色痰：肺炎链球菌肺炎特征性表现
- 血痰/咯血：需警惕肺结核、肺癌、支气管扩张

### 就医指征
- 咯血或痰中带血
- 咳嗽伴体重明显下降（警惕肺癌）
- 咳嗽伴夜间盗汗、低热（警惕肺结核）
- 慢性咳嗽>8周，常规治疗无效
- 咳嗽伴呼吸困难

## 三、胸痛

### 危及生命的胸痛（需紧急评估）

**急性心肌梗死（AMI）：**
- 典型表现：胸骨后压榨性疼痛，向左肩、左臂、下颌放射
- 持续时间>30分钟，含服硝酸甘油不能完全缓解
- 伴出汗、恶心、濒死感
- **处理：立即拨打急救电话，嚼服阿司匹林300mg（无禁忌症情况下）**

**主动脉夹层：**
- 突发撕裂样剧烈胸痛，向背部放射
- 两侧血压差>20mmHg
- 需急诊CT血管造影确诊

**肺栓塞：**
- 突发呼吸困难伴胸痛
- 常有下肢深静脉血栓史（长途飞行、卧床后）
- 可有咯血

**张力性气胸：**
- 突发剧烈胸痛伴呼吸困难
- 患侧呼吸音消失
- 需紧急穿刺减压

### 非紧急胸痛

**胸膜炎：**
- 随呼吸加重的锐痛
- 可闻及胸膜摩擦音

**肌肉骨骼性胸痛（最常见）：**
- 局部压痛（+）
- 与体位、运动相关

**胃食管反流：**
- 烧灼感，饭后平卧加重
- H2受体拮抗剂或质子泵抑制剂（PPI）有效

## 四、头痛

### 原发性头痛（无器质性病变）

**紧张型头痛（最常见）：**
- 双侧压迫性或紧箍感
- 轻中度强度，不影响日常活动
- 与压力、焦虑、颈肩肌肉紧张相关
- 治疗：对乙酰氨基酚或布洛芬；放松训练

**偏头痛：**
- 单侧搏动性疼痛，中重度
- 伴恶心、呕吐、畏光、畏声
- 部分患者有先兆（视觉闪光、暗点）
- 治疗：曲普坦类（急性发作）；预防用药（普萘洛尔、托吡酯）

**丛集性头痛：**
- 单侧眼眶周围剧烈疼痛
- 伴同侧流泪、鼻塞
- 成簇发作，每次持续15分钟～3小时

### 需警惕的红旗征（继发性头痛）
- 突发"霹雳样"剧烈头痛（"一生中最严重的头痛"）→ 警惕蛛网膜下腔出血
- 发热+头痛+颈部僵硬→ 警惕脑膜炎
- 头痛进行性加重，伴视乳头水肿→ 颅内高压
- 50岁后新发头痛→ 排除颞动脉炎、颅内占位
- 头痛伴神经系统症状（肢体无力、言语障碍）→ 排除脑卒中

## 五、腹痛

### 按部位快速定位

| 部位 | 常见病因 |
|------|---------|
| 右上腹 | 胆囊炎、胆石症、肝炎 |
| 上腹部 | 胃炎、消化性溃疡、急性胰腺炎 |
| 左上腹 | 脾脏疾病、胃体病变 |
| 右下腹 | 阑尾炎（最常见）、右侧卵巢囊肿、回盲部疾病 |
| 左下腹 | 乙状结肠憩室炎、左侧卵巢囊肿、便秘 |
| 脐周 | 早期阑尾炎、肠系膜缺血、肠梗阻 |
| 弥漫性 | 腹膜炎、肠梗阻、肠易激综合征 |

### 急腹症识别（需紧急就医）
- 突发剧烈腹痛，持续不缓解
- 腹肌紧张、板状腹（腹膜刺激征）
- 腹痛伴高热（38.5℃以上）
- 腹痛伴呕血或便血
- 腹痛伴血压下降、脉搏细速（休克表现）
