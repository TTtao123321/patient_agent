# 高血压与糖尿病管理指南

## 一、高血压

### 诊断标准
非同日3次测量，收缩压≥140 mmHg 和/或舒张压≥90 mmHg。

### 分级

| 级别 | 收缩压（mmHg） | 舒张压（mmHg） |
|------|-------------|-------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120～139 | 80～89 |
| 1级高血压（轻度） | 140～159 | 90～99 |
| 2级高血压（中度） | 160～179 | 100～109 |
| 3级高血压（重度） | ≥180 | ≥110 |

### 高血压急症
- 血压急骤升高，通常>180/120 mmHg
- 伴靶器官损害：高血压脑病、急性心力衰竭、主动脉夹层
- **需立即就医，务必在1小时内将血压降低20%～25%，勿降过快**

### 生活方式干预（所有高血压患者）
1. **减少钠盐摄入：** 每日食盐<6g（约一啤酒瓶盖），可降低血压2～8 mmHg
2. **体重管理：** BMI控制在18.5～24，腰围男<90cm，女<85cm
3. **规律运动：** 每周5次，每次30分钟中等强度有氧运动
4. **戒烟限酒：** 男性酒精摄入<25g/天，女性<15g/天
5. **心理平衡：** 避免过度紧张和情绪激动

### 常用降压药

| 药物类别 | 代表药物 | 适应人群 |
|---------|---------|---------|
| ACEI | 培哚普利、贝那普利 | 合并糖尿病、心力衰竭、蛋白尿 |
| ARB | 厄贝沙坦、缬沙坦 | 同ACEI，ACEI咳嗽者替换 |
| CCB（钙通道阻滞剂） | 氨氯地平、硝苯地平缓释片 | 老年、收缩期高血压 |
| 利尿剂 | 氢氯噻嗪、吲达帕胺 | 容量负荷过重、老年 |
| β受体阻滞剂 | 美托洛尔、比索洛尔 | 合并冠心病、心力衰竭、心律失常 |

**注意：** ACEI/ARB常见不良反应：ACEI可引起干咳（约10%～15%患者），可换用ARB。两者禁用于妊娠期妇女和双侧肾动脉狭窄患者。

### 血压控制目标
- 一般高血压患者：<140/90 mmHg
- 合并糖尿病：<130/80 mmHg
- 合并慢性肾病：<130/80 mmHg
- 老年患者（>65岁）：<150/90 mmHg，若能耐受可降至<140/90 mmHg

## 二、2型糖尿病

### 诊断标准（符合以下任一项）
1. 空腹血糖≥7.0 mmol/L
2. 口服葡萄糖耐量试验（OGTT）餐后2小时血糖≥11.1 mmol/L
3. 随机血糖≥11.1 mmol/L 伴典型糖尿病症状
4. HbA1c≥6.5%（需在有质量认证的实验室检测）

### 典型症状
"三多一少"：多饮、多尿、多食、体重下降
但很多2型糖尿病患者早期无明显症状，仅在体检时发现血糖升高。

### 控糖目标

| 指标 | 目标值 |
|------|------|
| HbA1c | <7.0%（一般目标）；<6.5%（年轻、病程短）；<8.0%（老年、低血糖高风险） |
| 空腹血糖 | 4.4～7.0 mmol/L |
| 餐后2h血糖 | <10.0 mmol/L |
| 血压 | <130/80 mmHg |
| LDL-C | <2.6 mmol/L（合并心血管病风险<1.8 mmol/L） |

### 生活方式管理
- **饮食：** 控制总热量，减少精制碳水（白米饭、白面包、含糖饮料）；增加膳食纤维（蔬菜、全谷物）；均衡分配三餐
- **运动：** 每周≥150分钟中等强度有氧运动，每周2～3次抗阻训练；避免久坐（每坐30分钟起身活动）
- **体重：** 超重/肥胖患者减重5%～10%可显著改善血糖

### 常用降糖药

**一线药（无肾功能禁忌均可使用）：**
- **二甲双胍：** 首选，500mg 每日2次，随餐服用；减少心血管事件，不引起低血糖；肾功能eGFR<30禁用

**合并心血管疾病或高风险患者优先选择：**
- **SGLT2抑制剂（如达格列净10mg 每日1次）：** 降血糖同时减少心衰住院和肾功能进展
- **GLP-1受体激动剂（如利拉鲁肽）：** 减重、心血管保护，注射剂型

**其他口服降糖药：**
- **DPP-4抑制剂（如西格列汀100mg 每日1次）：** 安全温和，不引起低血糖，适合老年患者
- **磺脲类（如格列美脲）：** 降糖强效但注意低血糖风险
- **α糖苷酶抑制剂（阿卡波糖）：** 餐时服用，降低餐后血糖为主

**胰岛素治疗适应症：**
- 口服药物控制不佳
- 急性并发症（DKA、高血糖高渗状态）
- 手术围术期
- 妊娠期糖尿病

### 低血糖识别与处理
**低血糖：** 血糖<3.9 mmol/L

**症状：** 心慌、手抖、出汗、饥饿感、头晕；严重时意识模糊、昏迷

**处理（15-15规则）：**
1. 立即进食15g快速吸收碳水化合物（3～4块葡萄糖片、150mL果汁、3块方糖）
2. 15分钟后再测血糖
3. 血糖仍<3.9 mmol/L，重复上述步骤
4. 血糖恢复后，若距下餐>1小时，再进食含淀粉的零食

### 糖尿病并发症筛查（每年一次）
- 糖尿病肾病：尿微量白蛋白/肌酐比值
- 糖尿病视网膜病变：眼底检查
- 糖尿病神经病变：踝反射、振动觉、10g尼龙单丝检查
- 心血管病变：心电图、血脂、颈动脉超声
- 足部检查：每次就诊均应检查
